🤖 ChatBot App (Flutter)

    A Flutter-based ChatBot application with a clean UI, smooth navigation, and animated transitions.
    This project is my second step in Flutter & Dart, where I implemented predefined Q&A logic as a foundation for future AI-powered chatbot integration.

🌟 Features

   🗨️ Predefined Q&A chatbot logic (initial version)

   🎨 Clean & minimal UI with VelocityX

   ⏱️ Chat timestamps using intl

   🔄 Smooth animated navigation & transitions

   📂 Clean coding practices & scalable structure

🛠️ Tech Stack

   Flutter

   Dart

   VelocityX   (UI toolkit)

   intl   (date & time formatting)

   Cupertino Icons

🔮 Future Improvements

    🤖 Integrate API/AI for dynamic chatbot responses

    🎙️ Add voice input & speech-to-text

    📱 Deploy to Android & iOS stores
